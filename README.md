# Self-Hosted Container Platform on AWS using Terraform and Docker

> A production-inspired DevOps project that provisions AWS infrastructure using Terraform and deploys a self-hosted container platform using Docker Compose.

---

## 📖 Project Overview

This project demonstrates how to provision cloud infrastructure on AWS using Terraform and deploy a self-hosted container platform using Docker.

The platform will include:

- Gitea (Git Server)
- PostgreSQL Database
- Nginx Reverse Proxy
- Private Docker Registry

The goal of this project is to showcase practical DevOps skills including Infrastructure as Code (IaC), containerization, networking, and deployment automation.

---

## 🚀 Technology Stack

- Terraform
- AWS
- Ubuntu Linux
- Docker
- Docker Compose
- Nginx
- Gitea
- PostgreSQL
